+MaheshKumar Tamboli  You can use the screen size as an additional selector. The Nexus 10 is "xlarge", while the Galaxy S3 is "normal". But I recommend to use the "smallest width" screen configuration for the layout buckets. The Nexus 10 falls into "sw600dp", while the Galaxy S3 falls into "sw360dp". See http://developer.android.com/guide/practices/screens_support.html#NewQualifiers
